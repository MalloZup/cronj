# cronj

A simple to use, cron-inspiried task scheduler

[![Build Status](https://travis-ci.org/zcaudate/cronj.png?branch=master)](https://travis-ci.org/zcaudate/cronj)

## Installation:

In project.clj, add to dependencies:

      [im.chit/cronj "1.4.0"]
